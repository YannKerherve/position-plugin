TCP Installation Guide

1. Download

First, download the latest version of the Windy Plugin for your operating system.

2. Run the Plugin

Extract the folder and run the TCP Windy bridge 2.0 (or the binary for your OS).

If a firewall popup appears, allow access to Private Networks.

3. Input Data

In the "Connection Settings" tab, enter your TCP source info:

4. Transmit

Click on the big "START TRANSMIT" button.

The status light should turn GREEN if the plugin is receiving NMEA data correctly.

5. It's Live!

Open Windy.com.

Your boat icon should now be visible, moving in real-time with all your onboard data layers active.

6. Using OpenCPN as a Server

To send data from OpenCPN to this plugin:

  1. Go to OpenCPN Options > Connections.
  2. Click Add Connection.
  3. Select Network and TCP.
  4. Check Output on this port.
  5. Address: 0.0.0.0 | Port: 10110.
Note: In the Windy Plugin (Step 3), use the IP 127.0.0.1 if OpenCPN is on the same computer.
